뼈대 만들기

"일단 화면을 띄워보자"
⏱️ 10분
📋 AI에게 보낼 프롬프트
동국제강 원료기획팀에서 사용할 업무 대시보드 웹 페이지를 만들어줘. 하나의 HTML 파일로 만들어줘. 상단에 진한 남색 배경의 헤더를 넣고, 왼쪽에 "DK 동국제강" 로고 텍스트, 오른쪽에 오늘 날짜와 "동국제강 원료기획팀 | 이돈석 팀장님" 텍스트를 넣어줘. 헤더 아래에 탭 메뉴 5개를 만들어줘: - 부재료실적 모니터링 - 등급별현황/수입관리 - 거래처 관리 - 공지사항 - 사용자관리 탭을 클릭하면 해당 내용이 보이고, 다른 탭 내용은 숨겨지게 해줘. 지금은 각 탭에 "여기에 OO 내용이 들어갑니다"라고만 표시해줘. 전체 배경은 밝은 회색(#F4F6F9), 헤더는 진한 남색(#1B2A4A), 포인트 색상은 오렌지(#E67E22)로 해줘.

✅ 확인 체크리스트

화면에 네이비색 헤더가 보이나요?
5개 탭을 클릭하면 내용이 바뀌나요?

💬 강사 한마디

지금 결과물은 HTML(구조) + CSS(디자인) + JavaScript(탭 전환 동작)의 조합입니다. AI가 세 가지를 한번에 만들어준 거예요!

숫자 카드 넣기

"핵심 지표를 한눈에"
⏱️ 10분
📋 AI에게 보낼 프롬프트
첫 번째 탭 "부재료실적 모니터링"에 내용을 추가해줘. 맨 위에 숫자 카드 4개를 가로로 나란히 배치해줘: 1. 연간 목표: 272.5만 톤 (설명: 2024년 연간 계획 합계) 2. 누계 실적: 281.5만 톤 (옆에 "초과" 표시를 오렌지색으로) 3. 계획 대비 달성률: 103.3% (옆에 "초과" 표시) 4. 거래처 평균 성과율: 90.4% 카드는 흰색 배경에 둥근 모서리, 큰 숫자는 굵고 크게, 설명은 작은 회색 글씨로 해줘.

✅ 확인 체크리스트

4개 카드가 가로로 나란히 보이나요?
숫자가 크고 눈에 잘 들어오나요?

💬 강사 한마디

이런 숫자 카드를 "KPI 카드"라고 합니다. 대시보드에서 가장 중요한 요소예요. 경영진이 한눈에 현황을 파악할 수 있게 해줍니다.

차트 넣기

"숫자를 그림으로"
⏱️ 15분
📋 AI에게 보낼 프롬프트
KPI 카드 아래에 차트 2개를 나란히 넣어줘. 차트는 Chart.js 라이브러리를 CDN으로 불러와서 사용해줘. 왼쪽 차트 (너비 60%): "월별 계획 대비 실적" - 막대 차트 - 가로축: 1월~12월 - 주황색 막대: 실적 (만톤 단위로 표시) - 파란색 테두리 막대: 계획 - 제목 옆에 "12월 누계 103.3%" 표시 아래 샘플 데이터를 사용해줘: 계획(만톤): 21.5, 21.5, 22.5, 24.5, 24.5, 25.0, 24.0, 20.0, 20.5, 20.0, 25.0, 23.5 실적(만톤): 23.9, 20.6, 22.9, 21.4, 22.6, 25.6, 27.1, 24.7, 22.5, 25.2, 23.1, 21.8 오른쪽 차트 (너비 40%): "월별 주요 거래처 납품 추이" - 꺾은선 차트 - 거래처 3개: 일보앤틱(파란색), 대한강철(주황색), 동국자원(초록색) - 적당한 샘플 데이터를 넣어줘 화면이 좁아지면 차트가 위아래로 배치되게 해줘.

✅ 확인 체크리스트

막대 차트에서 주황(실적)과 파란(계획)이 구분되나요?
꺾은선 차트에 3개 거래처가 각각 다른 색으로 보이나요?

💬 강사 한마디

Chart.js는 인터넷에서 무료로 가져와 쓸 수 있는 차트 도구입니다. CDN이라는 방식으로 인터넷 주소 한 줄이면 사용 가능해요. AI가 알아서 연결해줬습니다.

엑셀 데이터 붙여넣기 ⭐

"진짜 우리 데이터로!" — 오늘의 하이라이트
⏱️ 20분
📋 AI에게 보낼 프롬프트
차트 아래에 "월별 계획 대비 실적 데이터" 영역을 추가해줘. 이 영역의 핵심 기능: 엑셀에서 데이터를 복사해서 붙여넣으면, 위의 차트와 KPI 숫자가 자동으로 바뀌는 기능이야. 구체적으로 이렇게 만들어줘: 1. "붙여넣은 값 적용" 버튼과 "저장" 버튼을 만들어줘 2. 버튼을 누르면 큰 텍스트 입력창(textarea)이 나타나게 해줘 3. 사용자가 엑셀에서 아래 형태의 데이터를 복사해서 붙여넣어: 4줄짜리 데이터 (탭으로 구분됨) 1줄: 인천 계획 숫자 12개 (1월~12월) 2줄: 인천 실적 숫자 12개 3줄: 포항 계획 숫자 12개 4줄: 포항 실적 숫자 12개 4. "적용" 버튼을 누르면 붙여넣은 텍스트를 줄바꿈과 탭으로 나눠서 숫자로 변환하고, 인천+포항 합계로 차트와 KPI 카드를 자동 갱신해줘 5. "저장" 버튼을 누르면 localStorage에 저장해서 새로고침해도 데이터가 유지되게 해줘 6. 페이지를 열 때 localStorage에 저장된 데이터가 있으면 자동으로 불러와줘 아래에 현재 데이터를 테이블로도 보여줘 (인천 계획, 인천 실적, 포항 계획, 포항 실적 4줄) 그리고 그 아래에 "거래처 납품 실적관리 데이터" 영역도 만들어줘: - 큰 텍스트 입력창에 엑셀 5개 열을 복사해 붙여넣기 (전기일 / 공급업체명 / 검수등급명 / 입고량 / 입고금액) - "원본 데이터 반영" 버튼으로 데이터를 파싱해서 테이블로 표시 - "저장" 버튼으로 localStorage에 저장 - 데이터가 많으니까 100개씩 페이지 나누기

✅ 확인 체크리스트

텍스트 입력창에 위 숫자를 붙여넣고 적용하면 차트가 바뀌나요?

검수 등급 매핑 관리

"품명을 등급으로 자동 변환!"
⏱️ 15분
📋 AI에게 보낼 프롬프트
"검수 등급 매핑 관리" 기능을 추가해줘. 이 기능의 목적: 거래처 납품 데이터의 "검수등급명(품명)"을 보고 자동으로 국고 등급(국고상/국고중/국고하/선반설)으로 분류하는 매핑 테이블을 관리하는 기능이야. 기본 매핑 데이터: - 국고상: 생철A, 생철B, 생철AL, 슈레더B - 국고중: 중량A, 중량AS, 중AL, 중량 ALC(가위), 중량BS, 모터블럭 - 국고하: 중량B, 경량A, 경량B, 경량L, 길로틴A, 길로틴B, 절단S, 압축A, 압축B, 슈레더C, 중량BLS, 중량C, 경량C, 경량T, 경량TC, 경량S, 중량BLC, 중BL - 선반설: 선반A, 선반C, 압축C, 압축D 구체적으로 이렇게 만들어줘: 1. 등급별로 그룹핑된 매핑 테이블을 보여줘 (국고상/국고중/국고하/선반설 각각 색상 구분) 2. 각 등급 그룹에 새 품명을 추가할 수 있는 입력란과 "추가" 버튼 3. 기존 품명 옆에 "삭제" 버튼으로 매핑 제거 가능 4. 품명의 등급을 다른 등급으로 이동할 수 있는 드롭다운 5. "검색" 입력란: 품명을 입력하면 어떤 등급에 매핑되어 있는지 바로 표시 6. 매핑되지 않은 품명이 납품 데이터에 있으면 "미분류" 경고 표시 7. localStorage에 저장해서 새로고침해도 매핑 데이터 유지 8. "초기화" 버튼으로 기본 매핑 데이터로 되돌리기 가능 9. 거래처 납품 실적 데이터와 연동: 납품 데이터의 검수등급명을 이 매핑 테이블로 자동 분류해서 등급별 입고량/입고금액 요약 표시

✅ 확인 체크리스트

등급별 매핑 테이블이 색상 구분되어 보이나요?
새 품명을 추가하고 삭제할 수 있나요?
품명 검색 시 매핑된 등급이 바로 표시되나요?
새로고침해도 매핑 데이터가 유지되나요?

💬 강사 한마디

실무에서 품명(검수등급명)은 수십 가지인데, 보고서에선 국고상/중/하/선반설 4개 등급으로 요약해야 합니다. 이 매핑 테이블을 만들어두면 데이터가 들어올 때마다 자동 분류가 되어 수작업을 줄일 수 있어요.

나머지 탭 채우기

"기능을 하나씩 추가"
⏱️ 20분
📋 프롬프트 A — 거래처 관리 탭
세 번째 탭 "거래처 관리"에 내용을 추가해줘. 상단에 KPI 카드 4개: 1. 등록 거래처: 8개사 2. 월 공급능력 합계: 11.5만 3. 금년 납품량 합계: 26.5만 4. 대표 거래처: 현대스크랩 아래에 거래처 목록 테이블을 만들어줘: 코드 | 거래처명 | 지역 | 대표자 | 연락처 | 월공급능력(톤) | 금년납품량(톤) | 신뢰등급 | 납품실적 | 관리 샘플 데이터 8개를 넣어줘. 신뢰등급은 A, B+, B 같은 형태로, 납품실적은 프로그레스바로 표시해줘. 관리 컬럼에 "수정", "삭제" 버튼을 넣어줘. "+ 거래처 등록" 버튼을 만들고, 클릭하면 입력 폼이 팝업으로 뜨게 해줘. 등록/수정/삭제 모두 localStorage에 저장해줘.
📋 프롬프트 B — 공지사항 탭
네 번째 탭 "공지사항"에 내용을 추가해줘. 왼쪽 절반: 공지사항 게시판 - 표: 구분(필독/일반) | 제목 | 작성자 | 작성일 - 샘플 공지 5~6개 넣어줘 (철스크랩 수급계획 관련 내용으로) - "+ 등록" 버튼을 누르면 공지 작성 팝업이 뜨게 해줘 오른쪽 절반: 팀 일정관리 - 이번 달 캘린더 (달력 모양) - "이전/다음" 버튼으로 월 이동 - "+ 일정 등록" 버튼 - 날짜 클릭하면 일정 추가할 수 있게 모두 localStorage에 저장해줘.

마무리 다듬기

🔥 문제 해결 프롬프트 모음

😵 화면에 아무것도 안 나와요

에러가 나요. 화면에 아무것도 안 보여요. 코드를 확인해서 고쳐줘.

📊 차트가 안 그려져요

차트가 안 그려져요. Chart.js CDN이 제대로 로드되는지 확인하고, 차트 생성 코드에 에러가 없는지 확인해서 고쳐줘.

📋 데이터 붙여넣기가 안 돼요

엑셀에서 복사한 데이터를 붙여넣고 적용 버튼을 눌러도 반응이 없어요. 탭으로 구분된 텍스트를 파싱하는 부분을 확인하고 고쳐줘.

🎨 디자인이 이상해요 (겹침/너무 작음)

카드들이 겹쳐 보여요 (또는 차트가 너무 작아요, 글씨가 안 보여요). CSS를 확인해서 레이아웃을 고쳐줘.

💾 저장이 안 돼요 (새로고침하면 사라짐)

저장 버튼을 눌러도 새로고침하면 데이터가 사라져요. localStorage에 저장하고 불러오는 부분을 확인해서 고쳐줘.

⭐ 보너스: 한 방에 전체를 만들고 싶다면?

시간이 부족하거나 단계별 실습이 어려우면, 이 프롬프트 하나로 전체를 만들 수 있습니다.

동국제강 원료기획팀용 업무 대시보드를 하나의 HTML 파일로 만들어줘. 서버나 데이터베이스 없이, 브라우저만으로 동작해야 해. 데이터 저장은 localStorage를 사용해줘. 차트는 Chart.js CDN을 사용해줘. 디자인: 네이비(#1B2A4A) 헤더, 오렌지(#E67E22) 포인트, 밝은 회색(#F4F6F9) 배경 상단 헤더에 "DK 동국제강" 로고, 날짜, 조회연도 선택, 팀명, 다크모드 토글, 로그아웃 5개 탭 메뉴: 1. 부재료실적 모니터링 - KPI 카드 4개 (연간목표 272.5만톤, 누계실적 281.5만톤, 달성률 103.3%, 거래처성과율 90.4%) - 월별 계획 대비 실적 막대차트 (1~12월, 실적=주황, 계획=네이비) - 월별 거래처 납품 추이 꺾은선 차트 (3개 거래처) - 핵심 체크 포인트 카드 - 엑셀 데이터 붙여넣기 기능: textarea에 탭 구분 데이터를 붙여넣으면 차트와 KPI 자동 갱신 (인천/포항 계획 실적 4줄 형태) - 거래처 납품 실적 데이터도 엑셀 5개열(전기일/공급업체/검수등급/입고량/입고금액) 붙여넣기 가능 - 100건 단위 페이지네이션 2. 등급별현황/수입관리 - KPI 카드 4개 - 월별 구매량 추이 복합차트 (막대+꺾은선) - 인천공장 배분 도넛 차트 (등급별) - 수입 계약/운송 현황 테이블 (상태 배지) 3. 거래처 관리 - KPI 카드 4개 + 거래처 목록 테이블 (8개 샘플) - 등록/수정/삭제 기능 (모달 팝업) 4. 공지사항 - 좌측: 공지사항 게시판 (등록 기능) - 우측: 팀 일정관리 캘린더 5. 사용자관리 - 사용자 목록 테이블 + 등록 기능 모든 데이터는 localStorage에 저장. 반응형 디자인 (넓으면 가로, 좁으면 세로). 하단 푸터: "Copyright(c) 동국시스템즈"
